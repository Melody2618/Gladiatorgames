**Gladiators Coliseum Simulator – Java**

Built a tournament simulation system using object-oriented design and Binary Search Trees to manage cities and duelers in a rule-based elimination game.

**Key Features:**

- Implemented BST for efficient city insertion, search, and removal

- Designed recursive duel-selection algorithms with priority rules

- Managed dynamic elimination of cities based on population state

- Parsed structured input files to construct full game environment

- Applied tree traversal and inorder successor logic for node deletion

- Modeled duel mechanics with winner reintegration and state updates

**Concepts Used:**
Binary Search Trees, Recursion, Tree Traversal, Inorder Successor, Object-Oriented Design, File Parsing, ArrayLists, Randomization
